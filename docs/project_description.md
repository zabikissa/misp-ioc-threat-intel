# Projet : Veille Cyber Automatisée avec MISP

## Description
Ce projet automatise la collecte et l’exploitation des **indicateurs de compromission (IOC)** depuis **MISP**. 
Il est destiné aux équipes **SOC, CERT, CSIRT et Threat Intelligence** pour améliorer la veille cyber et la réactivité face aux menaces.

## Fonctionnalités
- Connexion sécurisée à l’API MISP
- Extraction automatique des derniers IOC
- Filtrage selon le TLP (Traffic Light Protocol)
- Export CSV des IOC pour analyse ou intégration SIEM/SOAR
- Automatisation via cron pour mises à jour régulières

## Cas d’utilisation
- Alimenter une base de données interne de menaces
- Générer des rapports pour l’équipe de sécurité
- Déclencher des actions automatiques basées sur les IOC

## Avantages
- Réduction du travail manuel
- Réactivité face aux menaces récentes
- Centralisation et structuration des IOC
